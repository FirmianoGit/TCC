## Variações dos Operadores

### Crossover Types
1 = Order Crossover (OC) - Preserva ordem relativa
2 = Two-Point Crossover (TP) - Simples e rápido
3 = PMX Crossover - Mantém mapeamento
4 = PTL Crossover - Position-based com diversidade

### Mutation Types
1 = Swap - Troca dois elementos
2 = Insert - Move um elemento
3 = Multiple Swap - Múltiplas trocas
4 = Multiple Insert - Múltiplas inserções (RECOMENDADO)

## Recomendações

Para problemas com 10-20 jobs:
  → Usar Preset 3 (exploração local) ou Preset 5 (teste rápido)

Para problemas com 20-50 jobs:
  → Usar Preset 1 (balanceado) - padrão

Para problemas com 50-100 jobs:
  → Usar Preset 2 (exploração agressiva)

Para problemas com 100+ jobs:
  → Usar Preset 4 (convergência rápida) com população aumentada
